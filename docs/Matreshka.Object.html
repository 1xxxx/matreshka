Basic usage

new MK.Object;

Passing object

// creates Matreshka.Object instance with 2 enumerable keys (a, b)
new MK.Object({ a: 1, b: 2 });
// same as new MK.Object().jset({ a: 1, b: 2 });

	'extends': MK.Object,

jset(key, value, evtOpts) → {mkObject}

Sets given property and adds key to enumerable list

This is common method of Matreshka.Object insance that does two things:

1. Sets property.

2. Adds given key to enumerable list (key enumerates via Matreshka.Object#each method, includes in result object returned by Matreshka.Object#toObject and Matreshka.Object#toJSON).

Parameters:

value *

Examples

Basic usage

this.jset( 'a', 1 ).jset( 'b', 2 );

Matreshka.Object#each method example

this.jset( 'a', 1 ).jset( 'b', 2 );
// sets 'c' to 3 but not adds keys to enumerable list
this.set( 'c', 3 );
this.each( function( value, key ) {
	console.log( key, value ); 
});
// logs 'a' 1 and 'b' 2

Matreshka.Object#keys method example

this.jset( 'a', 1 ).jset( 'b', 2 );
// sets 'c' to 3 but not adds keys to enumerable list
this.set( 'c', 3 );
console.log( this.keys() ); // logs [ 'a', 'b' ]

Matreshka.Object#toObject method example

this.jset( 'a', 1 ).jset( 'b', 2 );
// sets 'c' to 3 but not adds keys to enumerable list
this.set( 'c', 3 );
console.log( this.toObject() ); // logs { a: 1, b: 2 }

After using Matreshka.Object#jset you can work with property as with regular ptoperty

this.jset( 'a', 1 ).jset( 'b', 2 ); // sets properties and adds to enumerable list
this.set( 'a', 3 ); 
this.b = 4;
