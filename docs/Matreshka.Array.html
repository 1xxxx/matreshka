Class: Array

Matreshka. Array

Matreshka Array class. Extends Matreshka.

new Array()

Source:
  • array.mk.js, line 81
Examples

Basic usage

new MK.Array;

Passing length

// creates Matreshka.Array instance with length = 42
new MK.Array( 42 );

Passing elements

// creates Matreshka.Array instance with length = 2
new MK.Array( 'Hi', { a: 'b' } );

Inheriting

var MyClass = Class({
	'extends': MK.Array,
	constructor: function() {
		// calls MK.Array constructor with this context and given arguments
		MyClass.parent.constructor( this, arguments );
	},
	method: function() {}
});
